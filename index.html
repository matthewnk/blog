Day One

The first day...

This site is, obviously, a complete mess. It’s a bit frunstrating because I know what I want to accomplish, but I’m just having a lot of trouble with the CSS. HTML makes sense, and so far, JavaScript is actually quite fun. EVEN GIT is starting to make sense! But CSS just seems like the most ridiculous system imaginable for styling and organizing a page. It feels like trying to do surgery with a pocket knife taped to the end of yard stick. To put it another way, I need to put some extra effort into getting a handle on CSS. To that end I’m going to start by re-watching the Treehouse video.
